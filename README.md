# Data Analyst Portfolio

Portfolio of SQL-based data analysis projects demonstrating business insights, product analytics, and data-driven decision making.

---

## Projects

### 1. Sales Analysis
- Analyzed customer revenue, profit, and ROI
- Identified high-value and high-efficiency customers
- Compared revenue vs profitability vs efficiency metrics
- Highlighted limitations of ROI when acquisition costs are excluded

---

### 2. User Behavior & Funnel Analysis
- Analyzed user conversion funnel across device types (mobile, desktop, tablet)
- Identified major drop-off between product view and add-to-cart stages
- Detected inconsistencies in event-level tracking (purchase without add-to-cart)
- Built session-based funnel model to improve accuracy
- Demonstrated how data structure impacts metric reliability
