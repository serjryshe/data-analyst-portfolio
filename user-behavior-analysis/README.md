# User Behavior & Funnel Analysis

## Objective
Analyze how users interact with a product across different devices and identify where users drop off in the conversion funnel.

---

## Dataset
- users
- sessions
- events
- products
- event_products
- subscriptions
- marketing_spend

---

## Key Questions
- How do users behave across mobile vs desktop?
- Where do users drop off in the funnel?
- Which device converts better?
- Are funnel metrics reliable?

---

## Approach
1. Built event-level funnel (initial analysis)
2. Identified inconsistencies in conversion metrics
3. Switched to session-level funnel modeling
4. Compared conversion rates by device
5. Interpreted results and data limitations

---

## Tools Used
- SQL (MySQL)
